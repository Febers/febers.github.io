Java 排序算法

2019.03.15 Febers 算法  热度 ℃

排序算法一直是程序员开发和面试的重点，本文将重点讲解几种常用的排序算法。

稳定性

对于一个数组a {6,2,4,6,1}，其中有两个值为 6，为 a[0] 和 a[3]，排序之后的结果有两种

1 2 4 6 6
a[4] a[1] a[2] a[0] a[3]
a[4] a[1] a[2] a[3] a[0]

如果排序结束之后，a[0] 可以保证一定在 a[3] 前面，即原有的顺序不变，则该算法属于稳定的排序算法

冒泡排序、基数排序、插入排序、归并排序、桶排序、二叉树排序等

否则，属于不稳定的排序算法
